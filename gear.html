Stuff I use

A list of the stuff that I use to get my work done. I also mantain a collection of retro Apple products.

2015 MacBook Pro

As of June 2023, this is my primary machine.

Apple Thunderbolt Display

27" 1440p - Still built better than most cheaper displays these days.

Apple USB Keyboard

My primary keyboard is currently a '98 Bondi Blue Apple USB Keyboard, which originally came with the first revision iMac G3, and the G3 Blue and White. This is only on my main setup temporarily as my Logitech G613 started to give up.

Logitech MX Master [1st gen]

This is one of the oldest devices still sitting on my main setup. Super comfortable to use, and the USB dongle works on all of my vintage Macs. So a win-win for me!

iPhone SE [2nd gen]

Currently running the iOS 17 Developer Preview. Love using StandBy!

AirPods Pro [1st gen - MagSafe case]

One of the best devices I have. Love using these both indoors and outdoors.
