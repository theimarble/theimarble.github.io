Screen Ruler 1.0 for Mac OS X

Download



Overall features:

- 64 bit

- HUD horizontal and vertical rulers

- Simple hide and show window

- Keyboard shortcuts for hide and show


Alignment guides and rulers are very convenient inside applications. If there is an application that doesn't have any of these capabilities, you can use this free, and simple, application to space out items in a canvas.
