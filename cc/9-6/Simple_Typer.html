Simple Typer 2.0 for Mac OS X

Download



Updates in 2.0

- New drawer interface for your editing options

- New background for the main window

- Added keyboard shortcuts

- Changed the styles of some buttons

- New location for the undo and redo buttons

- A minor printing bug has been fixed

- New application icon


Initial release features

- 64 bit

- Tab interface to hold buttons

- Controls to access fonts and colors

- Spell check

- Text to speak

- Printing capability

- Clipboard actions (cut, copy, paste, etc)

- Unique drag and drop save and load interface

- Convenient keyboard shortcuts


Simple Typer is an easy to use, 64 bit, word processor for Mac OS X. It includes quite a few editing features, a unique drag and drop save and open interface, printing capabilities, and more.
